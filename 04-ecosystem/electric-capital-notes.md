# Electric Capital Notes

Research notes about developer ecosystem measurement and its relevance to the Midnight Network.

Electric Capital publishes the annual **Developer Report**, which analyzes activity across crypto ecosystems using GitHub data.

Reference:
https://www.developerreport.com/

## How Developer Ecosystems Are Measured

Electric Capital analyzes several signals:

- Number of active GitHub repositories
- Monthly active developers
- Commit activity
- Contributor growth
- Cross-chain developer participation

These metrics help estimate the strength and growth of blockchain ecosystems.

## Why This Matters for Midnight

For emerging ecosystems such as Midnight, developer visibility is important.

Key signals include:

- Public GitHub repositories referencing Midnight
- Projects using Midnight-related technologies
- Research and documentation about the network
- Developer experimentation with Compact and privacy infrastructure

Growth in these signals may influence how the ecosystem is perceived by investors, researchers, and developers.

## Ecosystem Observation

Tracking GitHub activity related to Midnight can provide early insight into:

- developer adoption
- ecosystem expansion
- research interest
- long-term network viability

This repository documents observations related to Midnight's developer ecosystem.
